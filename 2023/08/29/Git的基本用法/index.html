【Git】基本用法

Author: SANSHI Date: August 29, 2023  23:00:49 Category: git

Git&GitHub

终端的基本操作

  • pwd(print work direction)显示当前终端会话所在目录位置
  • ls(list file)显示当前目录下所有文件
  • cd(change direction)切换目录。后面跟路径“cd..”上一级。“cd “xxx(目录名)””是进入下一级。
  • git version确认安装版本号
  • git config –global user,name “sanshi”

config是配置，–global是全局的意思

这里跟的M表示与上次提交不一样，需要重新提交

git commit -m “第一次提交”

可以跳过vim编辑。

Git操作

新建本地仓库

打开对应文件夹，右击进入终端

  • git init
  • git clone https://github.com/SAN-SHIa/GitTry.git

提交代码三把斧

  • git add hello.txt
  • git commit

​ 此处需要注意先按“a/i”进入编辑模式，输出文件信息，后点击“esc”推出编辑。英文：后加wq正式退出。

  • git push 确认提交，然后即可在GitHub中查看到更新
